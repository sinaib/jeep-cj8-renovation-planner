What Is This

A personal restoration planner for a specific car —
a 1989 Jeep CJ8 Scrambler

This is a web app built to plan, track, and execute a full frame-off restoration of a 1989 Jeep CJ8 Scrambler that was partially disassembled 8 years ago and left unfinished.

The core idea: a restoration this complex needs an advisor that knows the car — not a generic todo list. The AI understands the CJ8's specific failure points, sourcing challenges in Israel, and how the phases of work depend on each other.

It's a single-user, offline-first app. No backend. No accounts. Everything lives in the browser. The only network call is to the Anthropic API for the advisor.

The vehicle
Jeep CJ8 Scrambler · 1989
AMC 258 4.2L inline-6 · Dana 300 transfer case · 103.5" wheelbase. Partially disassembled. Parts both on and off. Stalled mid-project.
The goal
Take it from current state (stalled disassembly) to reliable daily driver / occasional off-road. DIY everything except welding. Budget in ₪, parts sourced in Israel first.
The context
Based in Israel. Parts cost 30–40% premium vs US. Primary local supplier: Jeepland (searchable via API). All costs in ₪.
Why an app, not a spreadsheet
✓ The advisor knows what's in the plan — asks it questions and it gives answers based on your actual tasks
✓ It catches gaps proactively — CJ8 failure points you wouldn't think to add get flagged automatically
✓ Parts lists, step-by-step guides, cost estimates — all attached to the right task, not in a separate doc
✓ Photos attached to tasks — send a photo of a rusted frame rail and get a condition assessment

Background Intelligence

Three silent analysis triggers

Beyond the conversation, the advisor runs proactive analysis automatically — no message required. All three triggers are silent: they call tools, update the plan, but never add messages to the conversation.

On plan change
Debounced Gap Scan
After any phase or task change, waits 8 seconds then scans for CJ8-specific gaps if 10+ minutes have passed since last scan.
🔍
Checks for CJ8 known failure points with no corresponding task
⚠
Flags safety items that are missing from the plan
🔗
Detects uncaptured dependency chains
🔕
Silent — never adds conversation messages, only flag_gap calls
On task completion
Post-Completion Analysis
Immediately when a task is marked done, analyzes what just changed and prepares the next tasks.
✅
Identifies tasks now unblocked by this completion
🔩
Populates parts lists for newly-unblocked tasks if missing
📋
Writes step-by-step guides into next tasks
📝
Adds follow-up notes based on what was just done
On app load (weekly)
Weekly Health Check
Runs once per week on load. A holistic audit that checks the plan's completeness and health without any user action.
⏰
Flags stale "active" tasks with no recent notes
📦
Checks for over-loaded phases (>10 tasks)
🗂
Detects missing phases for a complete CJ8 restoration
🔕
Only calls flag_gap, never creates tasks
Embedded Technical Knowledge

The CJ8 context the advisor carries

The system prompt embeds deep CJ8-specific knowledge so the advisor never gives generic advice. Every recommendation is filtered through the reality of a 1989 Scrambler.

🔧 CJ8 Mechanicals

Engines: AMC 150 2.5L (base) or AMC 258 4.2L inline-6
Transmission: T4/T5 4-speed manual, Dana 300 transfer case
Axles: Dana 30 front, AMC Model 20 rear (known weak point for shaft breakage)
Frame: ladder-type, prone to rust at body mounts and rear crossmember
Wheelbase: 103.5" (vs CJ7's 93.4") — bed adds weight affecting suspension
CJ8 body parts significantly harder to source than CJ7

⚠ Known Failure Points

Rear main seal, oil pan gasket, head gasket
Carter YF carburetor — flooding and idle issues
Brake master cylinder — known failure after standing
Steering box wear
AMC 20 rear axle shafts — break under off-road stress
Wiring harness brittleness after 35+ years
Soft top and door weatherstripping deterioration

🇮🇱 Israel-Specific Context

Parts cost 30–40% premium vs US pricing (customs, VAT)
Local: JEEPOLOG, Jeepland (searchable), 4x4 Israel
US: Quadratec, Morris 4x4, Omix-ADA, Crown Automotive, RockAuto
Customs clearance strategy for US orders
All costs discussed in ₪ (Israeli Shekels)

📚 Embedded Manuals

Battlefield Repairs Manual (TM 9-2320-356-BD) — US Army field repair guide with emergency procedures, improvised repairs, field diagnostics
JEEPOLOG CJ8 Manual — Hebrew military maintenance manual: CJ8-specific torque values, fluid capacities, maintenance intervals, brake specs, electrical rebuild procedures
Technical Stack

How it's built

A focused React SPA — no backend, all state on-device. The only network calls are to the Anthropic API.

UI Framework
React 19 + Vite 8
TypeScript throughout. Framer Motion for animations. All inline styles — no CSS framework.
State
Zustand + localStorage
Persisted store with versioned migrations. Schema version 2 with deduplication on load.
File Storage
IndexedDB
Photo binaries stored in IndexedDB. Lightweight metadata index in Zustand store. No file size limits.
AI
Claude claude-sonnet-4-6
Streaming responses via Anthropic API. Tool use with 26 tools. Vision for photo analysis. Direct browser → API.
Testing
Vitest + Testing Library
329 tests across 10 test files. Store logic, tool handlers, component rendering, migration correctness.
Parsing
oxc (via Vite)
Fast Rust-based parser. Requires standard TypeScript patterns — no TypeScript generics in forwardRef calls.
No backend required. Your API key lives in .env.local. All data is local to your browser. Exporting the full plan (JSON) is available via Settings → Export.
